# GabrielAdikwu
An innovative web application showcasing events, team, and mission information for Gabriel Adikwu.

![Build Status](https://img.shields.io/badge/build-status-success-green) ![License](https://img.shields.io/badge/license-undefined-lightgrey) ![Language](https://img.shields.io/badge/language-typescript-blue)

## Description
GabrielAdikwu is a web application built with TypeScript that serves as a platform to present detailed information about events, team members, and organizational vision and mission. This application enhances user engagement and accessibility to essential organizational content, supporting community interaction and awareness.

## Features
- Responsive design optimized for various devices
- Dynamic event showcasing with images and descriptions
- Comprehensive team profiles with mission statements
- SEO-friendly structure and implementation
- Easily maintainable and extendable TypeScript codebase

## Tech Stack
- TypeScript
- Next.js
- React
- CSS (with PostCSS)
- ESLint

## Getting Started

### Prerequisites
- Node.js (v12 or later)
- npm or yarn package manager

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/JesseJohn7/GabrielAdikwu.git
   cd GabrielAdikwu
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
   or using yarn:
   ```bash
   yarn install
   ```

## Usage
To start the development server, run:
```bash
npm run dev
```
or using yarn:
```bash
yarn dev
```
Open your browser and navigate to `http://localhost:3000` to view the application.
